
# Synthetic Integrity Digital Twin (PIML Framework)

This repository generates a synthetic refinery-like digital twin with:

- 3D equipment visualization
- Piping attribute dataset
- Corrosion-related inputs
- Structured export in TSV format

All data is synthetic and intended for research and prototyping purposes only.

## Installation

pip install -r requirements.txt

## Run

python synthetic_unit_piml_pinn.py

Outputs will be generated inside the /data and /outputs folders.
